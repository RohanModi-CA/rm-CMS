# Configuration

We slowly are trying to make this configurable. We suggest putting any files you reference within the `user/` subdirectory within this config. Any paths you give in the `config.toml` are relative to that subdirectory, unless you use an absolute path, of course. 

---

- HeaderPath: The path to a HTML file that contains whatever you want to put in the `<head></head>`. **Do not** include `<head></head>`, or `<!DOCTYPE HTML>`, etc.
